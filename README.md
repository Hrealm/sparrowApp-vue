## Build Setup

``` bash
# clone project
git clone git@github.com:Hrealm/vueTrip.git

# enter project folder
cd vueTrip

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

```
## Start Local Server

```bash

# start node server at localhost:8008
node ./src/node/node.js

```
